# my-pet-bot-
synchronous telegram bot
Это мой первый телеграмм бот, который позволяет пользователям входить в аккаунт и работать с виртуальным балансом.

Стэк технологий:
-pyTelegramBotAPI (для работы с Telegram Bot API)
-pymysql (для подключения к MySQL)
-python-dotenv (для управления переменными окружения)
-email-validator (для проверки email)
-hashlib + secrets (для хеширования паролей)
-Хеширование паролей (SHA-256 + salt)

Возможности бота:
1. Регистрация
2. Авторизация по логину и паролю
3. Хранение паролей с солью и SHA-256
4. Отображение баланса
5. Меню

Структура проекта:
main.py — здесь прописана вся логика работы бота.
database.py — тут подключение и запросы mySQl
hashing.py — файл, отвечающий за хеширование паролей
.env.example  - пример настроек  окружения
read.me — описание бота

Установка и развертывание проекта:
1. Скопируйте себе репозиторий
2. Установите зависимости, которые написаны в requirements.txt
3. Настройте .env на основе своих данных:
	API_KEY=ваш_токен_бота
	DATA_BASE_HOST=localhost
	DATA_BASE_LOGIN=user
	DATA_BASE_KEY=password
	DATA_BASE_NAME=my_pet_bot_db
4. Запустите бота: python main.py
